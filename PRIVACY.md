# Privacy Policy

GitHub Authorized Apps - Inline Permissions is a read-only browser extension.

## Data collection

The extension does not collect, sell, share, or transmit personal data to the developer or any third party.

## What the extension reads

The extension runs only on GitHub settings pages under `https://github.com/settings/*`.

When enabled, it fetches the GitHub app detail pages that your signed-in GitHub session can already access. It reads the permission text on those pages and displays that text inline on the app list.

## Local storage

The extension stores one setting in browser sync storage:

- Whether inline permissions are enabled.

It also caches fetched permission text in session storage for the current browser session. This cache stays in your browser and is cleared by the browser when the session ends.

## Network access

The extension only makes same-site requests to GitHub pages needed to display authorized app permissions. It does not contact any non-GitHub server.

## Changes

Privacy policy changes will be published in this repository.
