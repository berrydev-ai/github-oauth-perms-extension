# Security Policy

## Supported versions

Only the latest version on `main` is supported.

## Reporting a vulnerability

Please report security issues privately by opening a GitHub security advisory for this repository.

Do not include:

- GitHub session cookies.
- Personal access tokens.
- Private repository names unless they are required to explain the issue.

## Scope

This extension is read-only. It fetches GitHub settings pages that the signed-in user can already view and renders permission text inline. It does not revoke access, grant access, call GitHub APIs, or send data to external services.
